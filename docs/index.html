Backup & Restore

All data lives in this browser's localStorage only. Clearing browser data, switching devices, or using a different browser will result in data loss. Export a JSON backup regularly.

💾 Export Full Backup (JSON)

Downloads all customers, suppliers, invoices, and expenses as a single JSON file.

📥 Import / Restore (JSON)

Restores data from a previously exported JSON backup. This will overwrite all current data.

🗑️ Clear All Data

Permanently deletes all data from localStorage. Make sure you have a backup first!
