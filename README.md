# 💰 Accounting Software

A free, open-source, **fully client-side accounting web app** — no server, no backend, no sign-up required. Runs entirely in your browser and stores data locally.

🌐 **Live App:** [andy-and-terry.github.io/accounting-software](https://andy-and-terry.github.io/accounting-software)

---

## ✨ Features

- 💵 **Income & Expense Tracking** — Log transactions with date, description, amount, and category
- 📊 **Dashboard** — At-a-glance summary of income, expenses, and net balance
- 📁 **Custom Categories** — Organise your finances your way
- 📄 **Transaction History** — Filterable and sortable transaction list
- 💾 **Local Storage** — All data is saved in your browser, no account needed
- 📤 **Export to CSV** — Download your data as a spreadsheet at any time
- 🌙 **Dark / Light Mode** — Easy on the eyes, day or night
- 📅 **Monthly Reports & Charts** — Visualise your spending over time

## 🚀 Getting Started

### Use Online (Recommended)
Just visit the live app — no installation needed:
👉 [andy-and-terry.github.io/accounting-software](https://andy-and-terry.github.io/accounting-software)

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/andy-and-terry/accounting-software.git
   ```
2. Open `index.html` in your browser — that's it!

## 🛡️ Privacy

All your financial data stays **on your device**. Nothing is ever sent to a server. Data is stored using your browser's `localStorage`.
